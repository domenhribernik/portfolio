01 The concept

One runner,
mirrored in code.

A digital twin is a living software copy of something real. Virtual Runner is exactly that for a person who runs: their motion, their data and their performance, captured from the phone in their pocket and rebuilt as a system you can watch, query and even play against.

The twist is that it was never one program. The thesis stitches together eight separate courses from my degree into a single working pipeline, each course contributing the part it taught me to build.

  • React
  • Node.js
  • MongoDB
  • Kotlin
  • Python
  • Java / libGDX
